# Title

筆算ドリルの問題印刷ボタンと解答印刷ボタンの分離

## Status

in-progress

## Background

筆算アプリで印刷ボタンが 1 つだけだと、問題と解答を別々に印刷したい用途に合わない。

## Scope

- docs/hissan/index.html の印刷ボタンを問題用と解答用に分ける
- クリックしたボタンに応じて印刷対象ページだけを出力する
- 既存の画面プレビューと印刷前後レイアウト切り替えを維持する

## Non-Scope

- 問題生成ロジックの変更
- 印刷レイアウト全体の再設計
- プレビュー画面の大幅な構成変更

## Acceptance Criteria

- 問題を印刷ボタンで問題ページだけが印刷対象になる
- 答えを印刷ボタンで解答ページだけが印刷対象になる
- 画面上の既存プレビューは崩れず、印刷後に通常表示へ戻る

## Steps

1. 既存の印刷ボタンと印刷対象 DOM を分離可能な形に整理する
2. ボタンごとの印刷対象制御を追加し、既存の印刷レイアウト処理と両立させる
3. ローカルで印刷起動フローを確認し、計画と結果を更新する
