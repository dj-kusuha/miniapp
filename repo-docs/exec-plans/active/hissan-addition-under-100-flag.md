# Title

筆算ドリルの足し算100以下フラグ追加

## Status

in-progress

## Background

筆算アプリの足し算で、答えが 100 を超えない問題だけを作りたい要望がある。

## Scope

- docs/hissan/index.html に足し算専用の「答えが100を超えない」設定を追加する
- 設定が有効なとき、足し算の問題生成結果を 100 以下に制御する
- 条件を満たせない桁数では設定を使えないようにする

## Non-Scope

- 引き算や掛け算の出題条件変更
- 印刷レイアウトや問題数の変更
- 筆算 UI 全体の再設計

## Acceptance Criteria

- 足し算を選んだときだけ新しい設定が表示される
- 設定を有効にすると、生成される足し算の答えが 100 以下になる
- 100 以下にできない桁数の組み合わせでは、設定が無効化される

## Steps

1. 既存の設定 UI と足し算の問題生成処理を確認する
2. 足し算専用フラグと有効/無効制御を追加する
3. 生成結果と HTML 構文を確認し、計画を完了状態へ更新する

## Decision Log

- 2026-04-29: 3 桁を含む組み合わせでは 100 以下の足し算が成立しないため、フラグを無効化して扱う

## Validation

- 実行前確認として既存の lint / test / build 設定が見当たらないことを確認した
