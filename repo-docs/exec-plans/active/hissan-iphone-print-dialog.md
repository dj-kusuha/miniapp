# Title

筆算ドリルの iPhone 再印刷時ダイアログ抑制

## Status

in-progress

## Background

iPhone で筆算アプリの「印刷する」を複数回使うと、2 回目以降に「この Web サイトから自動的に印刷することは禁止されています。」というダイアログが表示され、印刷を続けにくい。

## Scope

- docs/hissan/index.html の印刷起動処理の見直し
- iPhone 系端末でユーザー操作起点の印刷として扱われやすくする調整

## Non-Scope

- 問題生成ロジックの変更
- 印刷レイアウト自体の大幅な再調整

## Acceptance Criteria

- iPhone 系端末で 2 回目以降の印刷時にも自動印刷禁止ダイアログが出にくくなる
- 既存の印刷前後レイアウト切り替えが維持される
- 他端末の印刷動線を不必要に壊さない

## Steps

1. 現在の印刷ボタン押下から `window.print()` までの非同期処理を整理する
2. ユーザー操作の同期文脈で印刷を起動できるように処理を調整する
3. 画面表示と印刷起動が崩れていないかを確認し、計画を更新する

## Decision Log

- 2026-04-19: iPhone の再印刷時ブロックは `window.print()` が user gesture から外れて呼ばれている可能性を優先して対処する

## Validation
