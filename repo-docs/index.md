# Repo Docs Index

このディレクトリは、リポジトリ内でエージェントと人間が共有する記録システムです。
docs/ は GitHub Pages 用の公開コンテンツに予約しています。

## 入口

- ../AGENTS.md: 最初に読む短い地図
- ../ARCHITECTURE.md: 構造と境界の方針
- ../FRONTEND.md: 小さい Web アプリ向けの UI 実装指針
- ../PLANS.md: 実行計画の運用ルール

## サブディレクトリ

- design-docs/: 長期的な設計原則と設計メモ
- product-specs/: アプリごとの仕様
- exec-plans/: 進行中と完了済みの作業計画
- references/: 外部資料の要点や抜粋
